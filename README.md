# Invoice & PDF Generator

A professional web-based invoice and quote generator that creates branded PDFs in one click. Built for small businesses and freelancers who want professional invoices without paying monthly SaaS fees.

![Invoice Generator Screenshot](screenshot.png)

## Features

- Generate professional invoices and quotes as PDF
- Live PDF preview — updates as you type
- Upload your business logo
- Custom brand colors and fonts
- Supports 170 international currencies
- Automatic tax calculation
- Auto-numbered invoices (INV-001, INV-002...)
- Notes and payment terms field
- Works on desktop and mobile
- No subscription fees — yours forever

## Tech Stack

- Next.js 15
- TypeScript
- @react-pdf/renderer
- Tailwind CSS

## Getting Started

1. Clone the repo
```bash
git clone https://github.com/sharmakamayani/invoice-pdf-generator.git
cd invoice-pdf-generator
```

2. Install dependencies
```bash
npm install --legacy-peer-deps
```

3. Run the development server
```bash
npm run dev
```

4. Open [http://localhost:3000](http://localhost:3000)

## How to Use

1. Fill in your business details on the left panel
2. Add your client details
3. Add line items with quantity and rate
4. Choose your currency and tax rate
5. Upload your logo and pick your brand color
6. Click **Generate Preview** to see your PDF
7. Click **Download PDF** to save it

## Customisation

Each client gets their own branded version:
- Their logo and colors
- Their business name and details
- Their preferred currency
- Their payment terms
